Fixed Point

fixed point theorem
  1. If g is a continuous function on the interval [a, b] and for all x ∈ [a, b] it holds that g(x) ∈ [a, b] then g has a fixed point on [a, b].
  2. If, furthermore, for all x ∈ (a, b) it is fulfilled that g′(x) exists in (a, b) and |g′(x)| ≤ k < 1, then g has a unique fixed point p in [a, b].
