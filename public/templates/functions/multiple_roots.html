Multiple Roots

Definition:

We say that xv is a root of multiplicity m of f if and only if f (x) can be written as

f(x) = (x−xv)^m*q(x)


where q(xv) ≠ = 0. If m = 1, it is called a simple root.

theorem:

Let f ∈ C^m[a, b]. The function f has a root of multiplicity m in xv if and only if 0 = f(xv) = f′(xv) = f′′(xv) = · · · = f^(m−1)(xv) but f^m(xv ) ≠ = 0.
